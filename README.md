# Cloud Connected Environmental Data Logger

## Overview

This project is developed using the LPC2148 ARM7 microcontroller to monitor environmental conditions and upload sensor data to the ThingSpeak cloud using the ESP01 Wi-Fi module.

## Features

* LM35 Temperature Sensor
* MQ2 Gas Sensor
* ESP01 Wi-Fi Module
* ThingSpeak Cloud Integration
* 16x2 LCD Display
* EEPROM Data Storage
* UART Communication
* I2C Communication
* ADC Interface
* External Interrupt
* 4x4 Keypad
* Buzzer Alert

## Hardware Used

* LPC2148 ARM7 Development Board
* LM35 Temperature Sensor
* MQ2 Gas Sensor
* ESP01 Wi-Fi Module
* 16x2 LCD
* AT24C EEPROM
* 4x4 Keypad
* Buzzer

## Software Used

* Embedded C
* Keil uVision
